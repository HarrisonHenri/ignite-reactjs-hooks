## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/ignite-reactjs-hooks.git
```

```
$ cd ignite-reactjs-hooks
```

Install all the required dependencies

```
$ yarn
```

Then execute

```
$ yarn start
```
